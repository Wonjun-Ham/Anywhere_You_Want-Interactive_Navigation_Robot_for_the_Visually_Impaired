# Anywhere_You_Want-Interactive_Navigation_Robot_for_the_Visually_Impaired
시각장애인 분들은 낯선 곳을 혼자 찾아가는 게 불가능합니다. 여태까지의 주행 보조도구들은, 장애물 회피를 돕는 용도로, 길 안내 기능은 매우 제한적입니다. 이 프로젝트에서는 2024 ICRA best paper NoMaD를 변형해서, 길찾기 앱 화면과 전방 카메라 데이터를 이용해 주행을 결정하는 AI를 설계했습니다. 이를 통해 사용자가 로봇을 잡고 따라가기만 하면 목적지에 도착할 수 있도록 로봇을 제작하고 있습니다.
